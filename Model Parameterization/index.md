# What is the Maximum Species Age Output Extension?

The maximum species age extension outputs maps of the maximum ages for select tree/shrub species and the maximum age for all species combined.

# Release Notes

- Latest release: Version 3.0 — September 2018

- Full release details found in the User Guide and on GitHub.
- [View User Guide](https://github.com/LANDIS-II-Foundation/Extension-Output-Max-Species-Age/blob/master/docs/LANDIS-II%20Output%20Max%20Species%20Age%20v3.0%20User%20Guide.pdf).

# Requirements

You need:

- The [LANDIS-II model v7.0](http://www.landis-ii.org/install) installed on your computer.
- Example files (see below)

# Download

- The latest version can be [downloaded from GitHub](https://github.com/LANDIS-II-Foundation/Extension-Output-Max-Species-Age/blob/master/deploy/installer/LANDIS-II-V7%20Output%20Max%20Species%20Age%203.0-setup.exe). To install it on your computer, launch the installer.

# Example Files

LANDIS-II requires a global parameter file for your scenario, and separate parameter files for each extension.

Example files can be [downloaded from GitHub](https://downgit.github.io/#/home?url=https://github.com/LANDIS-II-Foundation/Extension-Output-Max-Species-Age/blob/master/testings/version-tests/Core7-MaxSppAge3.0/Output-maximum-species-age.zip).
